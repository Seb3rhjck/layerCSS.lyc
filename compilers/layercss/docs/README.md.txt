# LayerCSS (.lyc)

LayerCSS es un lenguaje de estilos modular y ligero basado en CSS.

## Características
- Modularidad con bloques como `@variables`, `@base`, `@component`, etc.
- Optimización automática del CSS generado.
- Compatible con proyectos frontend modernos.

## Instalación
1. Clona este repositorio.
2. Ejecuta el compilador con Node.js.

## Ejemplo
Ver el archivo `examples/example1.lyc`.
